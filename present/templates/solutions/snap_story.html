The title of this puzzle refers to the fact that all of these images are created using Snapchat filters and add-ons. A snap story is just a series of images someone can post on their own personal Snapchat. Clearly, this is a “cute boys” puzzle.

The images are shown in a grid ordered left to right, top to bottom, and they are alphabetized by the last name of the pictured boy. Looking at all the images, there are 16 different and distinct adaptations done to the 70 images. Each one is used between 3 and 7 times. Each of these forms a set, where a set is a group of men who were all on the same TV series, in the same movie (series), or in the same musical group without overlaps. However, each group has a member that doesn’t belong in it, and one of its members can be found in another group. These missing members form a 16 group loop. Using the size of the group as an index into the SURNAME of the member in the wrong set spells out a message.

  • Beverly Hills 90210 - Brian (Austin) Green - Group Size = 4 - E
  • One Direction - Liam Payne - Group Size = 4 - N

Since it forms a loop, you have to find the starting point which is Beverly Hills 90210. Reading the letters in order gives ENDER WIGGIN ACTOR which, with only one film version of Ender’s Game yields a final answer of ASA BUTTERFIELD for the puzzle.
