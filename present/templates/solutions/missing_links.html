The index gives an alphabetical list of golf courses which then have images presented in the rest of the brochure. The courses are:

  • Augusta National
  • Barnbougle Lost Farm
  • Cape Kidnappers
  • Cypress Point
  • Fishers Island
  • Kingston Heath
  • Merion
  • Mile Square
  • Pebble Beach

  • Royal Dornach (Struie)
  • Royal Melbourne
  • Royal Porthcawl
  • Royal Portrush (Dunluce)
  • Royal St. George’s
  • Shinnecock Hills
  • St. Andrews (The Old Course)
  • Winged Foot

In each image, hole numbers have been removed and a set of up to 5 numbers have been added in their place. The first step is to identify the correct hole numbers. Extract the letter for the correct hole number from the name of the course.

Royal Portrush (Dunluce) : 13,2,6 : HOP

Cypress Point : 5,4,9,6,7 : EROSS

Royal St. George's : 9,1,11,13 : ERRE

Pine Valley : 9,5,4 : EVE

Fishers Island : 3,1,10,11 : SFLA

Merion : 1,2 : ME

Mile Square : 6,7,4,10 : QUEE

St. Andrews (The Old Course) : 4,1,2,3,6 : NSTAR

Barnbougle Lost Farm : 6,15,2,13 : OFAS

Kingston Heath : 2,11,9,7 : IAHO

Shinnecock Hills : 8,10,6 : OKE

Royal Porthcawl : 8,13,10 : RWH

Winged Foot : 2,10,3,5 : ITNE

Royal Dornach (Struie) : 3,11,4,8 : YCAR

Royal Melbourne : 6,7,13,8,11 : MENLU

Pebble Beach : 10 : C

Augusta National : 11,1,8,7,5 : IANAS

Cape Kidnappers : 14,9,5 : SAK

Parsing all of the letters in order gives a set of famous gemstones. Extract a letter based on the month for which the type of gem is the birthstone.

  • E - HOPE - diamond (4)
  • R - ROSSER REEVES - ruby (7)
  • N - FLAME QUEEN - opal (10)
  • I - STAR OF ASIA - sapphire (9)
  • E - HOOKER - emerald (5)
  • E - WHITNEY - alexandrite (6)
  • L - CARMEN LUCIA - ruby (7)
  • S - NASSAK - diamond (4)

This gives the answer ERNIE ELS.
