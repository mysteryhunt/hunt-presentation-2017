This puzzle consists of a set of images labeled with numbers, which compose 11 different connect-the-dots series. Solvers need to first figure out which images belong in the same series based on visual similarities - each series consists of portions taken from the same photograph. Each connect-the-dots series spells a letter, and ordering these letters by the height of the "1" dots in each series produces the hint STATA CENTER, which indicates that solvers should head to the Stata Center to continue solving the puzzle.
Once in the Stata center, solvers will likely notice that the images are taken from various large canvases from Yann Arthus-Bertrand's "La Terre vue du ciel" photography series. Solvers need to repeat the connect-the-dots process to produce new letters, but this time using the locations of the selected images as they appear in the original photographs. If you order these new letters based on the conspicuous TVDC number label at the bottom of each canvas.
The letters spell out MARY POPPINS, which is the answer to this puzzle.
