For each of the given enumerations, five of the letter sequences below can be combined to form a clue of that enumeration. Each clue contains the word "this" and seems to reference something not given in the clue.

For each clue, the lengths of the five pieces combined, in order, form the zip code for an American city, and the name of that city is what the clue is referring to. For example, the first clue, MADE-UP TOWN WHERE THIS OPERA'S SET, has pieces of lengths 4,8,7,4,5. 48745 is the zip code for Mikado, MI, so "this opera" refers to The Mikado, and the answer to the clue is Titipu.

The first letters of the clue answers, in order, spell TEAM FROM LEFTOVER CODE. There are five unused pieces, which start with the letters A through E, giving them a natural ordering: AIGN, BORETH, CENON, DSENS, EEHERE. (The letters other than those initial letters spell the message IGNORE THE NONSENSE HERE.) The lengths of these pieces, in this order, are 46556. This is the zip code for the University of Notre Dame, and the team from this university is the FIGHTING IRISH.
