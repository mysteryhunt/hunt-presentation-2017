While on the surface this looks like a puzzle about the bat-and-ball game cricket, it’s actually about the darts game cricket.

Each of the numbered entries represents a throw in a dart game. The field position corresponds to a number on a dart board. The number of runs is an index into the player's name. Taking the indexed letter for every entry spells the message: NOPE FIRST FIND OUT ALL HITS THAT SCORE THEN YOU HAVE TO IGNORE NEAR MISSES TO GET A CLUE SOLVING THE TASK THEN YOUR WORK DECODING THAT ACQUIRED STATEMENT.

This means that you have to take the letters from only the throws that would score in a game of darts cricket. Score the game according to the standard rules of darts cricket, and the letters from just the scoring throws gives the message: FINAL SCORE VIGENERE USING KEYWORD QUITE.

The final score of the game is 63-629. Converting each number in the score to letters gives FCFBI. When deciphered with a Vigenère cipher using the key QUITE, it becomes PIXIE, which is the answer.
